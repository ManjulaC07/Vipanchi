# 🍫 Choco Luv – Chocolate Website

## 📌 Overview

Choco Luv is a simple and modern chocolate e-commerce website built using HTML, CSS, and JavaScript.
It showcases different chocolate products, special deals, and allows users to add items to wishlist and cart.

---

## ✨ Features

* 🏠 Single page layout (Home, Store, Top Picks, Deals, Journey)
* 🔍 Search functionality for products
* ❤️ Wishlist system (saved using localStorage)
* 🛒 Cart system (add products)
* 🎯 Interactive UI (heart toggle, slider)
* 📱 Responsive design (mobile friendly)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Font Awesome (icons)

---

## 📁 Project Structure

* index.html → Main page
* style.css → Styling
* script.js → Functionality
* wishlist.html → Wishlist page
* cart.html → Cart page
* images/ → Product images

---

## 🚀 How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Explore the website
